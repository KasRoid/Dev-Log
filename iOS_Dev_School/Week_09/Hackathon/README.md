**_CardGame with LoginPage_** is an app for playing simple but addictive card game. Ranking board and few other card games are coming on the way.

---

## Technologies

This project is created with

1. Swift 5.2.4
    - UIKit
    - FireBase
    - JGProgress
    - Onboarding

- iPhone ONLY (currently)
